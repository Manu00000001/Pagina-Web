Imagenes

Sintaxis de imágenes HTML
La <img> etiqueta HTML se utiliza para incrustar una imagen en una página web.
Técnicamente, las imágenes no se insertan en una página web; las imágenes están vinculadas a páginas web. El etiqueta crea un espacio de espera para la imagen de referencia.
La <img> etiqueta está vacía, solo contiene atributos y no tiene una etiqueta de cierre.
La <img> etiqueta tiene dos atributos obligatorios:

  • src: especifica la ruta a la imagen
  • alt: especifica un texto alternativo para la imagen

Ejemplo:

Se agrega el texto para agregar una imagen:
<img src="url" alt="alternatetext">

El src atributo requerido especifica la ruta (URL) a la imagen.

Ejemplo
<img src="../Imagenes/Flor.jpg" alt="Flor">


El alt atributo requerido proporciona un texto alternativo para una imagen, si el usuario por algún motivo no
puede verla (debido a una conexión lenta, un error en el atributo src o si el usuario usa un lector de pantalla).

Ejemplo: <img src="../Imagenes/Flor.jpg" alt="Flor">


Puedes usar el style atributo para especificar el ancho y el alto de una imagen.

Ejemplo:

<img src="../Imagenes/Tigre.jpg" alt="Tigre" style="width:250px;height:150px;">


Como alternativa, puede utilizar las width teclas y heigh atributos

Ejemplo:

<img src="../Imagenes/Tigre.jpg" alt="Tigre" width="200" height="125">

Imágenes en otro servidor/sitio web
Algunos sitios web apuntan a una imagen en otro servidor.

Ejemplo:

<img src="https://imagen.com">

Imágenes animadas
HTML permite GIF animados:

Ejemplo:

<img src="../Imagenes/gato.gif" height="125" width="200">

Imagen como enlace
Para usar una imagen como enlace, coloque la etiqueta dentro de la etiqueta:

Ejemplo:

<a href="Hola.html">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;"></a>

Imagen flotante
Usa la propiedad CSS floatpara dejar que la imagen flote a la derecha o a la izquierda de un texto:

Ejemplo:

<img src="../Imagenes/Pato.png" alt="Pato" style="float:right;width:100px;height:80px;">

Este es un parrafo con una imagen flotante
